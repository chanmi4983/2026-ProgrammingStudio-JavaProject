주제 선택 이유

읽은 책이나 읽고 싶은 책을 따로 정리하지 않으면 책 제목이나 진행 상태를 잊어버리기 쉽다. 
그래서 책 정보를 등록하고 조회하고 수정하고 삭제할 수 있는 콘솔 기반 책 관리 프로그램을 만들려고 한다.
